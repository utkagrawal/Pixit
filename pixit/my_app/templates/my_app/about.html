The Story Behind Pixit

As an artist, I struggled to find the pixel art I needed. While conventional images were easy to find, the specific pixelated designs I wanted were hard to come by. This led me to develop an application that seamlessly transforms standard images into pixel art. It was a creative solution to a common problem in the artistic community, simplifying the process of accessing and manipulating visual content.

Technologies Utilized:

  • Python programming language
  • OpenCV library
  • Django web framework
  • HTML
  • CSS
  • Bootstrap framework
  • JavaScript
  • Procreate (for design and artwork)
  • Jinja
  • Linode Hosting Platform
